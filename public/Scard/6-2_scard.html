인공지능과 과학 탐구 요약

01. 빅데이터 (Big Data)

  • 정의:
    기존 방법으로 처리하기 힘든 대규모의 복잡한 데이터.
  • 3V 특징:
    규모(Volume), 속도(Velocity), 다양성(Variety).
  • 과학적 활용:
    인간이 찾기 힘든 복잡한 패턴이나 상관관계를 발견 (예: 기후 변화 예측, 단백질 구조 분석).

Self Check

Q. 인공지능이 빅데이터를 통해 찾아내는 것은?

데이터 속의 패턴(상관관계)

02. 머신러닝의 종류

  • 지도 학습 (정답 O):
    문제와 정답(레이블)을 주고 학습.
    $\rightarrow$ 분류(개/고양이), 회귀(온도 예측).
  • 비지도 학습 (정답 X):
    정답 없이 데이터의 특징만으로 학습.
    $\rightarrow$ 군집화(비슷한 것끼리 묶기).
  • 강화 학습:
    보상과 벌칙을 통해 최적의 행동 학습 (알파고).

Self Check

Q. 정답(레이블)이 있는 데이터를 이용하는 학습법은?

지도 학습

03. 데이터 전처리

  • 정의:
    수집한 데이터를 학습에 적합한 형태로 가공하는 과정.
  • 중요성 (GIGO):
    "쓰레기를 넣으면 쓰레기가 나온다 (Garbage In, Garbage Out)."
  • 하는 일:
    결측치(빈값) 처리, 노이즈(이상값) 제거, 정규화(단위 통일) 등.

Self Check

Q. AI 모델의 성능을 위해 데이터를 다듬는 과정은?

데이터 전처리

04. 학습과 검증

  • 데이터 분리:
    전체 데이터를 훈련 데이터(Training)와 평가 데이터(Test)로 나눔.
  • 과적합 (Overfitting):
    훈련 데이터에만 너무 과하게 맞춰져, 새로운 데이터(평가 데이터)에서는 성능이 떨어지는 현상.

Self Check

Q. 모델 성능 평가를 위해 훈련에 쓰지 않고 남겨둔 데이터는?

평가(테스트) 데이터

05. 과학적 활용 사례

  • 지구과학:
    기상 데이터 분석 $\rightarrow$ 태풍 경로 예측, 기후 변화 시뮬레이션.
  • 생명과학:
    단백질 구조 예측 (알파폴드), 유전체 분석, 신약 개발.
  • 물리학:
    입자 가속기 충돌 데이터에서 새로운 입자 흔적 분류.

Self Check

Q. 구글 딥마인드의 AI가 해결한 생물학 난제는?

단백질 입체 구조 예측

06. 한계와 윤리

  • 알고리즘 편향:
    학습 데이터가 한쪽으로 치우치면 AI의 판단도 공정하지 못함 (성별, 인종 차별).
  • 설명 가능성:
    AI가 왜 그런 결론을 내렸는지 설명하기 어려운 블랙박스 문제 존재.

Self Check

Q. 학습 데이터의 불균형으로 인해 발생하는 AI의 문제는?

편향성 (Bias)
