2-2. 별의 진화 요약

01. 별의 탄생

  • 성운의 조건:
    온도가 낮고, 밀도가 높은 곳.
  • 에너지원:
    중력 수축 에너지 (위치E → 열E).
  • 원시별 → 별:
    중심 온도가 1,000만 K에 도달하면 수소 핵융합 시작.

Self Check

Q. 원시별의 중심 온도를 높이는 에너지원은?

중력 수축 에너지

02. 주계열성의 특징

  • 수명: 별의 일생 중 약 90% 차지. (질량이 클수록 수명 짧음).
  • 크기 일정:
    내부 압력(팽창) = 중력(수축).
    → 정역학 평형 상태.
  • 에너지원:
    수소 핵융합 반응 (4H → 1He + Energy).

Self Check

Q. 주계열성의 크기가 일정한 이유는?

중력과 내부 압력이 평형을 이뤄서

03. 태양 질량 별의 최후

  • 경로:
    주계열성 → 적색 거성 → 행성상 성운 → 백색 왜성.
  • 생성 원소:
    적색 거성 중심에서 헬륨 핵융합으로 탄소(C), 산소(O)까지 생성.
  • 물질 방출:
    행성상 성운 단계에서 우주로 물질 환원.

Self Check

Q. 적색 거성 중심부에서 생성되는 주요 원소는?

탄소(C), 산소(O)

04. 무거운 별의 최후

  • 경로:
    주계열성 → 초거성 → 초신성 폭발 → 중성자별 / 블랙홀.
  • 내부 구조:
    양파 껍질 구조. 중심부에서 철(Fe)까지 생성.
  • 폭발 후:
    철보다 무거운 원소(금, 우라늄 등) 생성.

Self Check

Q. 별 내부 핵융합으로 생성 가능한 가장 무거운 원소?

철 (Fe)

05. 태양계 형성

  • 성운설:
    태양계 성운 수축/회전 → 납작한 원반 형성.
  • 형성 순서:
    원시 태양(중심) + 미행성체(원반) → 원시 행성 → 태양계 완성.
  • 핵심: 태양과 행성은 같은 성운에서 동시에 탄생함.

07. 지구의 형성과 생명체 탄생

  • 지구의 형성 과정:
    미행성체 충돌(열 발생) → 마그마 바다 → 핵/맨틀 분리(밀도 차) → 원시 지각 형성 → 원시 바다 형성.
  • 생명체의 탄생:
    초기에 강한 자외선으로 육지 생활 불가 → 바다에서 최초의 생명체 탄생.
  • 산소와 육상 진출:
    광합성 생물(남세균) 출현 → 바다/대기 산소(O2) 증가 → 오존층(O3) 형성 → 자외선 차단 → 육상 생물 출현.
