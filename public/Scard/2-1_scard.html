2-1. 원소 형성 요약

01. 기본 입자의 탄생

  • 빅뱅(Big Bang): 우주의 모든 물질/에너지가 한 점 → 대폭발 → 급격한 팽창 (온도/밀도 하강).
  • 기본 입자 형성:
    • 쿼크(Quark): 위(u), 아래(d) 등 6종류.
    • 경입자(Lepton): 전자(e) 등.
  • 양성자/중성자: 쿼크 3개가 결합.
    양성자(uud, +1), 중성자(udd, 0)

Self Check

Q. 양성자와 중성자를 구성하는 기본 입자는?

쿼크 (Quark)

02. 헬륨 핵의 생성 (3분)

  • 개수비 변화:
    초기 1:1 → 중성자가 양성자로 붕괴 → 양성자:중성자 = 7:1 (약 3분 경).
  • 핵융합:
    양성자 2개 + 중성자 2개 → 헬륨 원자핵(He2+) 생성.
  • 질량비 (빅뱅 증거 1):
    수소 : 헬륨 = 3 : 1
    예측값과 스펙트럼 관측값 일치.

Self Check

Q. 헬륨 핵 생성 직전 양성자와 중성자의 개수비는?

7 : 1

03. 원자의 생성 (38만년)

  • 시기/온도: 빅뱅 후 38만 년, 약 3,000K.
  • 사건: 원자핵이 전자를 잡아 중성 원자 생성.
  • 우주 배경 복사 (증거 2):
    • 전자의 방해 없이 빛이 퍼져 나감 (투명한 우주).
    • 현재 약 2.7K 파장으로 관측됨.
    • 펜지어스와 윌슨이 발견.

우주 배경 복사

04. 우주의 원소 분석

  • 스펙트럼 종류:
    • 연속: 광원에서 나온 빛이 분광기를 통과하면서 나타남 (무지개색).
    • 방출(선): 고온의 기체가 방출한 특정 파장의 빛 (밝은 선).
    • 흡수(선): 저온의 기체 통과 (검은 선).
  • 알 수 있는 것:
    선의 위치 → 원소의 종류
    선의 굵기 → 원소의 함량(질량비) : 수소와 헬륨이 3:1의 질량비

Self Check

Q. 별빛이 저온의 성운을 통과할 때 나타나는 스펙트럼?

흡수 스펙트럼

05. 별의 탄생과 융합

  • 탄생: 성운(저온/고밀도) → 중력 수축 → 온도 상승 → 1,000만K.
  • 주계열성: 수소 핵융합 (H → He).
    중력(수축) = 내부 압력(팽창) → 크기 일정.
  • 적색 거성: 중심부 수소 고갈 → 중력 수축(중력>내부압력) → 온도 상승 → 1억K → 헬륨 핵융합 (He → C, O).
    • 다음 단계의 핵융합이 일어날 만큼 온도가 높지 않아 중심부는 중력(수축),
    중심부 바깥은 팽창하여 물질을 외부로 방출

Self Check

Q. 태양 정도 질량의 별이 만드는 최종 원소는?

탄소(C), 산소(O)

06. 무거운 원소의 생성

  • 초거성 (질량 큰 별):
    탄소 → 네온 → 마그네슘 → 철(Fe)까지 생성.
    *철은 원자핵이 가장 안정하여 더 이상 융합 X.
  • 초신성 폭발:
    별의 붕괴와 폭발 에너지로 철보다 무거운 원소 생성 (금, 납, 우라늄 등) → 우주로 방출.
